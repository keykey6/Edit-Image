# 超级全能免费图像工具箱（前后端分离完整版）

## 项目介绍

对标市面所有付费修图、证件照、图片处理网站，打造**全功能、无付费、纯免费、本地处理、模块化独立功能**的前后端分离项目。所有功能各自独立页面、独立路由、独立算法接口，可单独使用也可全部集成，可直接部署成网站/打包桌面软件，适合毕设、商业项目、个人开源项目。

**总功能数量：22个独立核心模块（碾压市面普通修图项目）**

## 技术栈（稳定、易部署、适合毕设/商用）

- **前端**：Vue3 \+ Vite \+ Element Plus \+ Canvas \+ 图片预览裁剪组件

- **后端**：Python FastAPI \+ OpenCV \+ Pillow \+ RemBG \+ 图像AI算法

- **存储**：本地临时缓存，无需数据库，轻量化零配置部署

- **特色**：所有功能模块化、独立路由、可单独开启/关闭、支持批量处理、高清导出无水印
